## 🚀 功能列表

### 👤 自身
- **上帝模式 / 无限生命**: 永不死亡。
- **无限体力**: 无限奔跑。
- **穿墙飞行 (NoClip)**: 穿过任何墙壁自由飞行。
- **自定义属性**: 实时调整速度、力量、跳跃力、重力、投掷力和抓取范围。
- **无武器后坐力/冷却**: 无限制开火。
- **穿墙抓取**: 隔墙与任何物品交互。
- **RGB 玩家**: 彩虹颜色循环。
- **去除迷雾**: 所有关卡视野清晰。
- **自定义 FOV**: 调整视场角 60 ~ 120。
- **自动闪避**: 敌人接近时自动闪避，可配置触发距离。

### 👁️ 视觉 (ESP)
- **现代 UI**: 简洁直观的菜单设计。
- **敌人透视**: 通过墙壁看到敌人，显示方框、名称、距离和血量。
- **物品透视**: 追踪贵重物品，显示方框、价值和名称，支持最低价值过滤。
- **玩家透视**: 追踪队友/其他玩家，显示血量和距离。
- **撤离点透视**: 轻松找到出口。
- **Chams 着色**: 为物品和敌人定制 3D 着色效果。
- **地图信息**: 显示地图物品总价值及玩家状态列表。
- **追踪线**: 从屏幕底部画线连接到敌人、物品或玩家。
- **ESP 预设**: 6种一键预设方案（自定义 / 高价值 / 仅敌人 / 仅玩家 / 全部显示 / 隐身模式）。
- **迷你雷达**: 圆形雷达覆盖层，显示敌人（红）、玩家（绿）、物品（黄）和撤离点（蓝）。

### ⚔️ 战斗 & 玩家
- **玩家交互**: 对大厅中任何玩家造成伤害、治疗、击杀或复活。
- **传送玩家**: 将玩家传送到自己身边、彼此之间或"虚空"中。
- **社交**: 伪造名字或使用彩虹名称效果。

### 👾 敌人
- **敌人控制**: 致盲所有敌人或秒杀。
- **传送敌人**: 将怪物传送到指定玩家身边。
- **怪物生成器**: 生成游戏中任意敌人（需要房主/MasterClient）。

### 📦 物品
- **物品生成器**: 生成自定义价值的贵重物品（需要房主/MasterClient）。
- **物品传送**: 将所有物品拉到你的位置。
- **远程出售**: 无需返回飞船即可出售物品。
- **自动拾取**: 自动收集范围内的贵重物品，可配置拾取半径和最低价值过滤。
- **自动出售**: 将物品传送到撤离点，全自动出售。

### 🎮 创造模式
- **类 Minecraft 创造模式**: 一键开启上帝模式 + 飞行 + 穿墙 + 无限体力 + 穿墙抓取 + 无冷却 + 去雾。
- **飞行控制**: WASD 移动，空格上升，Ctrl 下降，Shift 加速。
- **状态恢复**: 退出创造模式时，自动恢复之前的设置。

### 🔀 传送+
- **准心传送**: 瞬间传送到你注视的位置。
- **撤离点传送**: 跳转到最近的撤离点。
- **随机传送**: 在范围内随机传送。
- **传送点系统**: 保存、加载和管理命名传送点。

### 🛠️ 杂项 & 服务器
- **服务器浏览器**: 强大的大厅搜索，支持区域、玩家数、隐藏满员大厅等过滤。
- **快捷键管理**: 将任意功能绑定到你习惯的按键。
- **配置系统**: 自动保存和加载作弊设置。
- **JSON 配置**: 导出/导入所有设置到 JSON 文件，便于备份和分享。
- **双语界面**: 一键切换中文和英文界面。
- **整蛊功能**: 崩溃大厅、假玩家生成器、NPC 刷屏。

## 🛠️ 环境要求
- 任意 Mono 注入器（如 [SMI](https://github.com/wh0am15/SharpMonoInjector)）
- 游戏 **R.E.P.O**

## 🏗️ 如何编译
1. 用 **Visual Studio** 打开 `.sln` 或 `.csproj`，或使用 `dotnet build`。
2. 将配置设为 **Release**。
3. 生成解决方案（`Ctrl+Shift+B` 或 `dotnet build "r.e.p.o cheat.sln" -c Release`）。
4. DLL 文件位于 `bin/Release/r.e.p.o cheat.dll`。

## 💉 如何注入
使用自带的 `start_and_inject.bat` 或你喜欢的注入器，参数如下：
- **命名空间 (Namespace)**: `r.e.p.o_cheat`
- **类 (Class)**: `Loader`
- **方法 (Method)**: `Init`
